SMS 2.0

A sample application that allows you to use SMS 2.0 service. It demonstrates:
  • SMS 2.0 API
To check this service you need 2 telephone numbers.
This sample application tries to perform the following actions:
  • Connect to the service
  • Add a contact with the second telephone number (it must be online for the sample to work properly)
  • Wait until it receives the message "bye", responding to every message with the same text plus " to you too"
  • Remove contact
  • Disconnect from service
