AutoWP::SendAutoWP

Sends a WapPush message to the user's own number

def SendAutoWP(self, login, pwd, url, text)

Parameters
login
  String with user's telephone number

pwd
  String with user's password

url
  String with message URL

text
  String with message text

Return
Code received from service
  • 0 - OK
  • 1 - Invalid user format
  • 2 - Invalid URL
  • 3 - Invalid pasword format
  • 4 - Access denied
  • 5 - User is not registered
  • 6 - Invalid password
  • 7 - User registration expired
  • 8 - Text format invalid
  • 9 - Communication not established
  • -1 - Request error, missing or mistyped parameter
