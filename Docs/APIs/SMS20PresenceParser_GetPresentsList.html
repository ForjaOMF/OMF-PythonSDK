PresenceParser::GetPresentsList

Gets the user's presence status list

def GetPresentsList(self, response)

Parameters
response
  String with XML data received from server

Return
Contact list of the user in a dict indexed by userId.

{'wv:6xxxxxxxx@movistar.es': ['T', 'John'], 'wv:6yyyyyyyy@movistar.es': ['F', 'Peter']}

Presence status can be T(rue) if the contacts is online and F(alse) if the contact is offline

Below you can see a sample code to traverse the list and print userId, presence status and nickname: 
  cl = pp.GetPresentsList(xml)
  for n in cl :
    if cl[n][0] == 'T' :
      print cl[n][1] + '(' + n + ') is ONLINE'
    else : 
      print cl[n][1] + '(' + n + ') is OFFLINE'
