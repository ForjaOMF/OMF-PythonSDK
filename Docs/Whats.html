What's Open movilforum SDK?

Open movilforum is an initiative that puts open APIs at independent developers' disposal that give them the capability to interact with different mobility services, with the aim of making easier to develop open-source mobile applications and mash’ups with other Internet services.

This SDK provides the needed APIs to embed those mobility services into your applications.
